# Background Image Prompts for BounceBlob

Format: WebP, 2048x512 (4:1), quality 85
Place files in: `/public/assets/backgrounds/`

## IMPORTANT NOTES
- The game mirrors every other segment, so images don't need to be perfectly tileable
- But: **avoid unique objects near the left/right edges** (tree stumps, large rocks, etc.) — these look weird when mirrored. Keep edges simple/organic.
- Export as **PNG with transparency** first, then convert to WebP
- The left and right edges should have similar content (same general height, same density)
- NO checkerboard pattern — real alpha transparency needed

## Conversion tips

Convert PNG to WebP:
```bash
cwebp -q 85 input.png -o output.webp
```

For images with transparency (far/mid/near layers):
```bash
cwebp -q 85 -alpha_q 90 input.png -o output.webp
```

Sky layers can be saved without alpha for smaller size.
